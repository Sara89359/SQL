🌟 Sobre este repositorio<br>
Este proyecto recoge mi proceso de aprendizaje de SQL, desde los conceptos más básicos hasta consultas más avanzadas.
Aquí iré subiendo:
- apuntes
- ejercicios
- prácticas
- pequeños proyectos
- consultas interesantes que vaya descubriendo
Mi objetivo es construir un espacio organizado y útil tanto para mí como para cualquiera que esté empezando en SQL.
